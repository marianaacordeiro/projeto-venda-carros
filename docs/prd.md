# Product Requirements Document (PRD) - Retron

**Autor:** Mariana Cordeiro  
**Projeto:** Retron - Catálogo de Veículos Clássicos

## 1. Descrição do Projeto

O **Retron** é uma plataforma web voltada para entusiastas e colecionadores de automóveis. O sistema resolve a necessidade de uma vitrine digital organizada onde usuários podem explorar um catálogo selecionado de veículos clássicos, acessar detalhes técnicos e manifestar interesse de compra através de um fluxo de cadastro integrado com validação de endereço.

## 2. Atores do Sistema

- **Visitante:** Usuário não autenticado que pode navegar pela Home, visualizar o carrossel de destaques e explorar a listagem de veículos.
- **Cliente:** Usuário que realiza o cadastro/login, permitindo o registro de interesse em modelos específicos e a persistência de seus dados de preferência.

## 3. Histórias de Usuário

1. **Como Visitante**, eu quero visualizar um carrossel de fotos na página inicial para conhecer os modelos em destaque. (ID 04)
2. **Como Visitante**, eu quero que o site seja responsivo para que eu possa pesquisar carros pelo meu celular ou tablet. (ID 02)
3. **Como Cliente**, eu quero preencher meu CEP no formulário de cadastro para que meu endereço (Rua, Bairro, Cidade) seja preenchido automaticamente. (ID 24)
4. **Como Cliente**, eu quero que meus dados de contato (E-mail e Telefone) sejam validados para garantir que a loja consiga retornar meu contato. (ID 12, 21)
5. **Como Cliente**, eu quero clicar em "Tenho Interesse" para que minha intenção de compra seja salva no sistema. (ID 22)
